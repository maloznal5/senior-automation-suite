### Tech Stack:
* **Language:** Python 3.10+
* **Framework:** Aiogram 3.x (Asynchronous)
* **Logic:** Finite State Machine (FSM)
* **Tools:** Telegram Bot API, Logging, File I/O

### Key Features:
* **Automated Booking:** Multi-step registration (Name -> Service -> Photo -> Time).
* **Admin Dashboard:** Two-way communication between Master and Client.
* **Order Logging:** Automatic saving of client data to local storage.
* **Smart UI:** Interactive Reply/Inline keyboards with dynamic price calculation.

### How to run:
1. Clone the repo.
2. Install requirements: `pip install aiogram`.
3. Set your token in `bot.py`.
4. Run: `python bot.py`.
